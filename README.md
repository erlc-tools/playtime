### 📦 Installation

Clone the repository and install dependencies:

```batch
git clone [https://github.com/fin-github/playtime.git](https://github.com/erlc-tools/playtime.git)
cd playtime
npm install
```

---
### ⚙️ Build
```bash
npx build
```

---
### 💾Configure .env
**Windows:**
```batch
copy .env.template .env
```
**Unix (linux)**
```bash
cp .env.template .env
```
**⚠️Configure .env to your liking**

---
### 🚀Run
```batch
npx build
npx start
```
